# endgame

A WebRTC-enabled 3D chess game.

## Development

- Install dependencies (`npm install`).
- Run gulp to build targets and start devserver.
- Run `gulp dist` to minify and build distributable.
- Run `firebase deploy` to deploy to Firebase.
